Pricing Rules

The Golden Formula

Price = config.fixedAmount
      = 109500 cents
      = $1,095.00

That's it. No markup logic, no multipliers, no cost-plus math. Just one hardcoded number.

What is IGNORED

  • PEN Retail Price ($895)
  • PEN Dealer Cost ($450)
  • Vehicle Make/Model/Year/Mileage
  • Vehicle Price
  • Customer Zip Code
  • Finance Terms

Configuration

Everything is controlled via application.yml.

# library/src/main/resources/application.yml products: cost: # ══════════════════════════════════════════════════════ # VLP Configuration - Simple fixed amount # ══════════════════════════════════════════════════════ vehicle-loss-protection: fixed-amount: 109500 # $1,095.00 (Care & Protection +$100) # ══════════════════════════════════════════════════════ # Comparison: VRC has overrides # ══════════════════════════════════════════════════════ vehicle-service-contract: markup-amount: 1000000 retail-price-override-for-states: - FL # Florida override

How to change the price?

1. Update the fixed-amount in the YAML file.
2. Deploy the service.
3. Run the admin endpoint /test/fi-Rate/update-cached-markup to update existing cache.
